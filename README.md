# MotherhoodTruth WordPress Theme

Custom WordPress theme for **The Truth About Motherhood**.

Tagline: Honest motherhood. Real stories. No filters.

This repository contains a standalone production theme folder:

```text
wp-content/themes/MotherhoodTruth/
```

## Safe migration notes

- This theme does not modify WordPress core.
- Existing posts, pages, categories, tags, media, and SEO plugin metadata remain in WordPress.
- Upload or deploy only the `MotherhoodTruth` folder to `wp-content/themes/`.
- Preview the theme in WordPress before activating on production.
